# タイピングゲーム (FE略語・ビジネス用語向け)

ブラウザ上で動作する **インストール不要のタイピングゲーム** です。  
単語やフレーズを完全一致で入力して、60秒間の成功文字数を競います。

---

## 🔹 遊び方

1. ブラウザで `index.html` を開く  
2. 表示された単語やフレーズを入力欄に入力  
3. 完全一致すると次の単語が表示され、成功文字数が加算されます  
4. タイマーが 0 になると終了、成功文字数が表示されます  
5. 「もう一回」ボタンでリスタート可能

---

## 🔹 カスタマイズ

- 単語リストを変更したい場合は、`index.html` 内の `text` 変数を編集  
- タイマー秒数は `timeLeft` の値を変更  
- CSS を変えるとフォントや見た目も自由に変更可能
